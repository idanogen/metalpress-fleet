API ואינטגרציות

Make Data Store API

Endpoint: https://us1.make.com/api/v2/data-stores/83526/data Headers: Authorization: Token {VITE_MAKE_API_TOKEN} Params: pg[limit]=100 & pg[offset]=0 (pagination) Response: { records: [{ key, data: { driverName, phone, model, monthlyUsage: "JSON string", ... } }], count, pg }

Pagination: 100 רשומות בכל קריאה, fetching כל הדפים עד שנגמר.

React Query cache: staleTime: 5 דקות. Fallback לנתונים סטטיים מ-src/data/vehicles.ts (103 רכבים).

WhatsApp Webhook

URL: https://hook.us1.make.com/piugtkez49mveettgmenuepb2v16w7pl Method: POST Format: { action, timestamp, reportMonth, reportYear, driver: { id, name, phone, whatsappId } } Phone format: 0523694547 → 972523694547@c.us (Green API)

Cooldown: 48 שעות per vehicle per month-year combo. נשמר ב-localStorage (מפתח: fleet-reminder-timestamps).

Vehicle Images CDN

תמונות רכב מ-imagin.studio עם מיפוי עברית→אנגלית (26+ מותגים). Cache ב-memory Map. Fallback לאייקון אפור.

תהליכים עסקיים

דיווח ק"מ חודשי

1
נהגים מדווחים ק"מ בחודש הנוכחי עבור החודש הקודם (דרך Priority / טופס חיצוני)
2
הנתונים נכנסים ל-Make Data Store (רשומה אחת לרכב, כולל מערך monthlyUsage)
3
הדשבורד שואב את הנתונים ומחשב: מי דיווח, מי לא, חריגות
4
איריס (מנהלת צי) שולחת תזכורות WhatsApp לנהגים שלא דיווחו

זיהוי חריגות (Anomaly Detection)

1
לכל נהג — מחושב ממוצע ק"מ חודשי (מתוך חודשים שדווחו, לא כולל חודש נוכחי)
2
אם הק"מ החודשי חורג ב-X% מהממוצע (ברירת מחדל: 30%) → מסומן כחריגה
3
סוגים: spike (מעל), drop (מתחת), chronic_no_report (3+ חודשים רצופים ללא דיווח)

מעקב תוקף ליסינג / רישוי

1
לכל רכב — מחשב ימים עד תפוגה של ליסינג ורישיון
2
דחיפות: פג תוקף (שלילי) → קריטי (≤30 יום) → אזהרה (≤60) → בקרוב (≤90) → תקין (>90)

הערות חשובות

לוגיקת חודשים — חשוב!

ברירת מחדל: חודש קודם. נהגים מדווחים בחודש הנוכחי עבור החודש שעבר. לעולם לא להציג חודש נוכחי כ"לא דווח" — הוא עדיין לא נגמר. חישוב: getMonth() הוא 0-based, מרץ=2, חודש קודם=פברואר=2.

monthlyUsage מגיע כ-string

השדה monthlyUsage מגיע מ-Make Data Store כ-JSON string, לא כאובייקט. צריך JSON.parse() בשכבת ה-API.

נתונים סטטיים = fallback

src/data/vehicles.ts מכיל 103 רכבים — משמש רק כשאין חיבור ל-Make API. אל תערוך אותו ידנית בייצור.

רכב "מלאי"

רכב עם driverName = "מלאי" הוא רכב ללא נהג מוקצה. מופיע רק בטאב "מלאי" של ניהול צי, לא בדשבורד.

Cooldown WhatsApp

48 שעות per vehicle per month-year combo. מפתח localStorage: {vehicleId}-{month}-{year}. ניקוי אוטומטי של timestamps ישנים.
